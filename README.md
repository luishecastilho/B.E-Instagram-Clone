# Back-end of Instragram clone
 API of a Instagram Clone

## Technologies:
* Node.js
* Adonis
* Postgres
* Docker

## Start:

* Create a database called: "instagram"
* Create your .env file
* Run "npm install"
* Run "adonis migration:run"
* Run: 'docker build -t instagram .'
* Run: 'docker-compose up'
